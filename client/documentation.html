Documentation

/getByTitleAuthor

Supported Methods: GET, HEAD
Query Params: Title (String, Required), Author (String, Required)
Returns: JSON object with all data about book
Example Usage:
Request: /api/getByTitleAuthor?title=schmingledork&author=johnny
Result: {"title":"SchmingleDork","author":"Johnny","year":343...}


/getByGenre

Supported Methods: GET, HEAD
Query Param: Genre (String, Required)
Returns: List of JSON objects with book info
Example Usage:
Request: /api/getByGenre?genre=classicliterature
Result: {"booksWithGenre":[{"author":"Jane Austen","country":"United Kingdom"...}...]}


/getByYear

Supported Methods: GET, HEAD
Query Params: Lower Year (Int, Required), Upper Year (Int, Required)
Returns: List of book titles
Example Usage:
Request: /api/getByYear?yearMin=-300&yearMax=2000
Result: {"booksWithYear":["The Divine Comedy","The Book Of Job","One Thousand and One Nights"...]}


/getAllEntries

Supported Methods: GET, HEAD
Query Param: None
Returns: List of all JSON objects with book info
Example Usage:
Request: /api/getAllEntries
Result: {"data":[{"author":"Chinua Achebe","country":"Nigeria","language":"English"...}...]}


/addBook

Supported Methods: POST
Body Param: Title (String, Required), Author (String, Required), Year (Int, Required), Genre (String, Required)
Creates: An object that has basic info about a book
Example Usage:
Request: /api/addBook
Body: {title: 'MyCoolTitle', author: 'Me!', year: 204, genre: 'Modernism'}
Result: {"title":"MyCoolTitle","author":"Me!","year":204,"genres":["Modernism"]}


/addRating

Supported Methods: POST
Body Param: Title (String, Required), Rating (Int, Required)
Creates: Adds an object property to an existing object
Example Usage:
Request: /api/addRating
Body: {title: 'MyCoolTitle', rating: 3}
Result: {"title":"MyCoolTitle","author":"Me!","year":204,"genres":["Modernism"],"rating":3}
